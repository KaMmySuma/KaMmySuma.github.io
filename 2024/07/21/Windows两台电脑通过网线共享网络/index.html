Windows两台电脑通过网线共享网络

2024/7/21 教程 网络共享

可以省一个路由器的特殊情况

  • 问题背景
  • 方法一：Windows网络共享
  • 方法二：网卡桥接
  • 参考

问题背景

  1. 有两台电脑：
  • 电脑A可以上网，并且有空闲网口
  • 电脑B无法上网，并且有空闲网口
  1. 需求：
  • 通过一根网线连接两台电脑的空闲网口，并使电脑A的网络共享给B，使得两台电脑都可以正常上网
  • B 希望可以通过网线连接 AB 空闲网口的物理方式实现共享 A 的网络

方法一：Windows网络共享

Windows自带了网络共享功能

  1. 进入电脑 A 的 网络适配器选项 配置界面
  2. 在上网的网卡上右键 -> 属性，进入共享选项卡
  3. 勾选允许其他用户通过此计算机的Internet 连接来连接
  4. 此处我当前环境有线网卡只有这一个，如果有多个的话会需要选择分享的目标网卡，这时候就需要选择和B连接的空闲网卡了
  5. 用网线连接 A 和 B
  6. 在电脑B上配置 IP 和 DNS 自动获取即可，如果自己没有动过此设置，一般默认就是自动获取
  7. 顺利的话 B 已经可以上网了

方法二：网卡桥接

创建一个虚拟网桥，不依赖Windows网络共享。但是有一个问题，创建网桥的电脑无法使用clash。

  1. 进入 A 的 网络适配器选项 配置界面
  2. 同时选中上网网卡和闲置网卡，右键 ->桥接
  3. 桥接后成为一个网桥，设置 B 的网段与DNS 和上网网卡相同 或 设置 B 的网络自动获取 IP 和DNS 即可
  4. 顺利的话 B 可以上网了，但是 A 无法使用clash
