  • MS-CEINTL.vscode-language-pack-zh-hans
  • wangzy.sneak-mark
  • Live Server

VSCode突出显示中文标点符号而不高亮汉字

  • 安装插件sneak mark
  • Windows下的默认插件目录为：
    C:\用户名\.vscode\extensions\wangzy.sneak-mark-0.0.6\out
  • 找到目录下的配置文件extension.js
  • 修改第十四行为：const charCodeRegEx = /(，|。|（|）|：|‘|’|“|”|？|！)/g;
  • 文件-首选项-设置，搜索Non Basic ASCII选择inUntrastedWorkspace（一般默认是它）
