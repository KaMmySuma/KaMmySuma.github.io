学习笔记-法律

1899/11/30 学习笔记 学习笔记

菜就多练

经济

  • 名义GDP：当年产量×当年价格

  • 实际GDP：当年产量×基期价格

  • 税收的三个基本特征

    1. 无偿性：税收的本质
    2. 强制性：无偿的强有力保障
    3. 固定性：强制和无偿的必然要求

  • 政策性银行

    1. 由政府创立，以贯彻政府的经济政策为目标，在特定领域开展金融业务的不以营利为目的的专业性金融机构。
    2. 目前我国的政策性银行有中国进出口银行、中国农业发展银行。
      • 1994年中国政府设立了国家开发银行、中国进出口银行、中国农业发展银行三大政策性银行，均直属国务院领导。2015年3月，国务院明确国开行定位为开发性金融机构，从政策银行序列中剥离。

刑法

  • 诈骗罪的诈骗对象必须是具有财物处置权的人。例如欺骗保安将别人寄存的财物骗走的不是诈骗罪，而是盗窃罪，因为保安没有该财物的处置权。

  • 盗窃罪是指以非法占有为目的，使用和平手段窃取财物的行为。

  • 侵占罪，是指将代为保管的他人财物非法占为己有，数额较大，拒不退还的，或者将他人的遗忘物或者埋藏物非法占为己有，数额较大，拒不交出的行为。上述保安如果把财物占为己有，那就是侵占罪。

  • 共同犯罪：

    1. 二人以上共同故意。需要注意的是，不要求相对方已经实际遭受权益的损害。二人以上共同过失犯罪不以共同犯罪处理，应当负刑事责任的，按照他们所犯的罪分别处罚。
    2. 教唆犯在共同犯罪中可能起主要作用，也可能起次要或辅助作用，因此教唆犯可能是主犯也可能是从犯。
    3. 从犯，可以从轻、减轻或免予处罚

  • 对于犯罪的外国人，可以独立适用或者附加适用驱逐出境。

  • 对判处管制的犯罪分子，依法实行社区矫正

  • 拘役的期限，为一个月以上六个月以下。判拘役由公安机关就近执行。被判处拘役的犯罪分子每月可以回家一天至两天；参加劳动的，可以酌量发给报酬。

  • 死刑除依法由最高人民法院判决的以外，都应当报请最高人民法院核准。死刑缓期执行的，可以由高级人民法院判决或者核准

  • 判处死刑缓期执行的，在死刑缓期执行期间，如果没有故意犯罪，二年期满以后，减为无期徒刑；如果确有重大立功表现，二年期满以后，减为二十五年有期徒刑；如果故意犯罪，情节恶劣的，报请最高人民法院核准后执行死刑；对于故意犯罪未执行死刑的，死刑缓期执行的期间重新计算，并报最高人民法院备案。

  • 非法占有所得财产应该退回而不是没收

宪法

  • 我国宗教工作的基本原则包括：公民有信仰宗教的自由；坚持实行独立办教的原则；坚持政教分离的原则；国家保护正常的宗教活动；积极引导宗教与社会主义社会相适应。

劳动法

  • 文艺、体育和特种工艺单位经未成年人的父母或者其他监护人同意，可以招用不满16周岁的专业文艺工作者、运动员

  • 用人单位自用工之日起即与劳动者建立劳动关系。试用期包含在劳动合同期限内，从用工之日起起算，无权由谁决定或约定

  • 试用期工资不得低于本单位相同岗位最低档工资或者劳动合同约定工资的百分之八十，并不得低于用人单位所在地的最低工资标准。

    • 也就是说，如果转正后工资为1w元，那么试用期工资应不低于8k

  • 试用期最长不超过六个月。劳动合同期限三个月以上不满一年的，试用期不得超过一个月；劳动合同期限一年以上不满三年的，试用期不得超过二个月；三年以上固定期限和无固定期限的劳动合同，试用期不得超过六个月。

  • 试用期只能约定一次，同一单位就算换岗位了也不能再有试用期。

民法典

  • 不满八周岁的未成年人为无民事行为能力人，由其法定代理人代理实施民事法律行为。无民事行为能力人实施的民事法律行为无效。

    • 不到8岁不能赠送财产，赠送行为无效
    • 无效的或者被撤销的民事法律行为自始没有法律约束力。

  • 八周岁以上的未成年人为限制民事行为能力人，实施民事法律行为由其法定代理人代理或者经其法定代理人同意、追认；但是，可以独立实施纯获利益的民事法律行为或者与其年龄、智力相适应的民事法律行为。

    • 到了8岁可以赠送财产，但赠送行为有没有效由爹妈说了算。取决于其法定代理人是否追认。

  • 狗咬人，狗主人必须赔偿是错误的，如果能证明是被咬人作死故意逗狗造成的可以不用赔。

  • 送养&收养

    • 送养：符合资格才能把孩子送给别人收养。下列个人、组织可以作送养人：（一）孤儿的监护人；（二）儿童福利机构；（三）有特殊困难无力抚养子女的生父母。
    • 收养：收养人应当同时具备下列条件（一）无子女或者只有一名子女；（二）有抚养、教育和保护被收养人的能力；（三）未患有在医学上认为不应当收养子女的疾病；（四）无不利于被收养人健康成长的违法犯罪记录；（五）年满三十周岁。

  • 婚姻

    1. 要结婚的男女要亲自到机关登记。不可以委托他人
    2. 婚姻自由，他人无权干涉
    3. 被迫胁、被对方隐瞒重大疾病的，可申请撤销婚姻。
      • 撤销不等于无效，向法院确认该婚姻无效的说法是错误的

  • 夫妻共同财产

    1. 工资、奖金、劳务报酬；
    2. 生产、经营、投资的收益；
    3. 知识产权的收益；
    4. 继承或者受赠的财产，但是以下规定的除外；
      1. 婚前财产；
      2. 因受到人身损害获得的赔偿或者补偿；
      3. 遗嘱或者赠与合同中确定只归一方的财产；
      4. 一方专用的生活用品；
      5. 其他应当归一方的财产。
    5. 其他应当归共同所有的财产。

  • 分割夫妻财产

    1. 除有约定的外，先将共同所有的财产的一半分出为配偶所有，其余的为被继承人的遗产。
      • 你死后，卡里100万老婆先拿一半，另一半才算你的遗产，老婆将继续参与剩下50W的分割。
    2. 第一顺序：配偶、子女、父母；第二顺序：兄弟姐妹、祖父母、外祖父母。继承开始后，由第一顺序继承人继承，第二顺序继承人不继承；没有第一顺序继承人继承的，由第二顺序继承人继承。
      • 假设父母早亡，你只有个女儿，那么剩下的50W，老婆和女儿各分一半。最终老婆到手75W，女儿到手25W

  • 宣告死亡

    1. 下落不明满四年，或者因意外事件而下落不明满二年的，可以申请宣告死亡
    2. 不同利害关系人同时申请失踪和死亡，只要符合死亡条件，就按死亡宣告
    3. 宣告死亡了，其实没死，不影响被宣告死亡期间实施的民事法律行为的效力。
      • 死亡有效，如果你又活了，财产被分了，有权撤销死亡宣告，请求返还财产，原物没有保存的给予补偿。

公文

  • 决议↓
    • 适用于会议讨论通过的重大决策事项
    • 特点
      1. 权威性，一经公布，全党、全国上下必须执行
      2. 程序性，必须按法定会议法定程序讨论
      3. 指导性
    • 成文时间：决议正式通过的日期。标题下小括号注明会议名称及通过时间，也可以只写年月日
  • 决定↓
    • 适用于对重要事项做出决策和部署、奖惩相关、变更或撤销不适当的决定事项
    • 特点
      1. 规定性
      2. 指令性
      3. 稳定性
    • 根据内容和作用进行分类：
      1. 指挥性决定。对重大事项和行动做出的安排
      2. 知照性决定。用于机构设置、人事安排、撤销下级不适当决定等重要事项的宣告
      3. 奖惩性决定。表彰或处分
  • 命令↓
    • 适用于公布法规规章、实行重大强制性措施、批准授予和晋升衔级、嘉奖有关单位和人员
      • 需要注意区分的是：命令只奖不惩；而决定和通报可奖可惩。
      • 奖励对象关键词：“急难险重”的工作任务
    • 特点
      1. 权威：只有国家主席、全国人大常委会及委员长；国务院、总理、各部委、各部长、各委员会主任；地方人大代表大会委员会、县级以上地方各级人民政府，才可以在法定权限内发布命令。其他任何单位和个人无权发布。
        强制力，无条件执行
      2. 强制：无条件执行
      3. 严肃：不得随意更改或处理
      4. 限定：只限于重大事件，一般公务工作不得使用
    • 主要分类
      1. 任免令。仅限中央层面，其他级别不得使用
      2. 发布令。主席发布法律
      3. 行政令。公布重大强制性措施，例如《事业单位会计准则》
      4. 嘉奖令。是最高级别的奖励类型，表彰超人。
  • 公告↓
    • 适用于向国内外宣布重要事项或法定事项
    • 特点
      1. 限定：高层行政机关才能发布
      2. 广泛：范围面向国内外
      3. 内容和传播方式具有新闻性
    • 按内容分类
      1. 宣布重要事项的公告
      2. 宣布法定事项的公告
      3. 专业性公告
  • 公报↓
    • 适用于公布重要决定、重大事项
    • 特点
      1. 庄严：内容多是党内外、国内外普遍关心的重要事项
      2. 形式多样：可以通过新闻媒体刊发
      3. 公开
    • 按内容分类
      1. 新闻公报。以新闻形式在新闻媒体发布，阅知无限制，要求及时、真实
      2. 联合公报。政党、国家、政府等同层面之间，就某重大事项会议协商一致后，双方联合签署发布的文件
        • 联合公报有一些双方认可、联合签署的条文，比一般新闻公报具有更多具体内容
      3. 会议公报。党和国家重要会议情况发布的公报。例如《中国共产党xx届xx委员会全体会议公报》
      4. 统计公报。国家和政府统计机关发布的国民经济、社会发展方面情况的综合公报
  • 通告↓
    • 适用于在一定范围内公布应该遵守或周知的事项
    • 特点
      1. 广泛：使用频率和使用单位广泛
      2. 法规性
      3. 专业性：公安、交通、税务等，带有专门性质
    • 分为法规性通告和知照性通告
      1. 法规性通告强制性措施较多，违反将处罚
      2. 知照性通告目的是通知事项，没有强制性措施，受众只要接受所通知的事实
  • 通报↓
    • 适用于表彰、批评、传达精神、告知情况
      • 例如《xxx特大交通事故的通报》
    • 特点
      1. 教育
      2. 说理
      3. 时效
    • 分类
      1. 表彰性通报
      2. 批评性通报
      3. 情况通报
  • 报告↑
    • 向上级汇报、反馈、回复上级询问
    • 特点
      1. 陈述
      2. 沟通
      3. 单向：上级无需批复报告，避免在报告中向上级请求批示
      4. 可以一文多事
  • 请示↑
    • 适用于向上级请求指示、批准
    • 特点
      1. 单一：一事一文，一文一事
      2. 期复：必须回复
      3. 时效：搞快点给我批复
  • 批复↓
    • 适用于答复下级请示
    • 特点
      1. 公文中唯一被动的文种。下级有请示，上级才有批复。
      2. 针对：请示什么就批复什么
      3. 指导性
  • 议案↔
    • 适用于向同级提出审议事项
    • 特点
      1. 发文机关：各级人民政府（向同级人大或人常）
      2. 行文定向流动
      3. 时限性极强
  • 函↔
    • 适用于不相隶属机关之间商洽工作、询问、答复、请求批准、审批事项
    • 特点
      1. 行文方向多样性
      2. 使用的广泛性
      3. 写作的灵活性
    • 按性质分类（西装和便装）
      1. 公函。用于机关单位正式公务活动来往
      2. 便函。日常事务工作处理
    • 按内容和用途分类
      1. 商洽函。不相隶属机关之间商谈事宜
      2. 询问答复函。对公务中不确定的问题提出咨询、给予答复。没有固定方向，有无隶属关系皆可使用
      3. 求批审批函。向别家的老大提出请求批复或反过来答复审批
    • 按行文方向
      1. 发函。发起方，结束语一般为：此函询、特此函达、如蒙惠允、不胜感荷。
      2. 复函。答复方，结束语一般为：复函口、特此函复、特此复函。
  • 纪要↑↓↔
    • 适用于记载会议主要情况和议定事项
    • 特点
      1. 纪实：如实反应会议基本内容事项，未讨论议题、未达成共识的事项不能写
      2. 纪要：对会议内容进行归纳总结，记录主要事项，突出会议重点和主题，而不是有闻必录
      3. 约束：与其他行政公文一样，具有一定约束力
  • 通知↓
    • 适用于发布、传达要求下级机关执行和有关单位周知或者执行的事项，批转、转发公文。
    • 对于表彰、批评、传达重要精神、重要情况，优先使用通报，而不是通知。
    • 特点
      1. 对于干部的任免或聘用等需有关单位及人员周知的事项，一律使用通知的文种
  • 意见↑↓↔
    • 适用于对重要问题提出见解和处理办法。
    • 特点
      1. 灵活：上下平行皆可
      2. 广泛：对行文机关没有限制，人民团体皆可
      3. 指示和参考：下行文具有指示性、上行文和平行文具有参考性
      4. 时限和连续：问题解决了，有关意见就自动失效；长期问题时效也长，还可以陆续出台补充修改意见。

  • 公告与公报的区别：

    1. 发文机关：
      • 公告仅限制高层机关才能发布
      • 公报是政党机关都可以使用
    2. 受文范围：
      • 公告多为某领域政策、知识。可以是国内外机构组织个人。侧重于国家机关向国内外宣布重要事项或法定事项。
        • 例如，中国政府网公告主要用于宣布重要法定事项、规章制度、指示规定等，旨在让受众群体知晓遵守执行。
      • 公报更广泛，没有特定性和固定性，面向社会大众广而告之。公报侧重内容都是重大决策决定、过程经过、成效成果，向社会公开报道，旨在使大众了解。
        • 例如，会议公报主要用于介绍重要会议情况和公布会议结果；各省区市的经济运行公报主要用于公布有关经济发展数据；科研机构的科学公报主要用于公布重大科学研究成果。
    3. 表达形式：
      • 公告篇幅短，简练准确、开门见山、直书其事。
        • 例如，《新疆维吾尔自治区统计局第七次全国人口普查公告》，开篇明义，即对全区14个地州市及所辖县市区人口数据进行公布。相较于公报，公告就事论事，简短精练，不作阐述评论。
      • 公报结构严谨、层次清晰，语言庄重热烈
        • 例如，《中国共产党新疆维吾尔自治区第十届委员会第一次全体会议公报》，标题含义完整，主体内容详细，包含时间、地点、参加人员、会议内容、发布机构、公布日期等，依次介绍会议宗旨、主要议题、重要决策、取得成果、决定事项等。从语言风格来看，用词既庄重又通俗，语气既严肃又热烈，对会议的重大意义和产生的影响等进行阐述或评论。

    • 也就是说，除非上级直接交办，否则不能给上级机关负责人（个人）报送，应该报送给上级机关（单位）

  • 处理公共危机的原则：

    1. 时间性原则：指反应速度。及时处理、及时控制。时间性原则是处理公共危机的最基本、最首要的原则。
    2. 效率性原则：指行动速度。快速反应，高效调动社会资源
    3. 协同性原则：指不同行业、不同部门协同一致从而应对危机
    4. 科学性原则：指专人专治。针对因工业技术而引起的灾害、由自然灾害而造成的危机事件。
