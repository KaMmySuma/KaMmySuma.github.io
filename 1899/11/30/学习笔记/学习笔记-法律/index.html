学习笔记-法律

1899/11/30 学习笔记 学习笔记

菜就多练

劳动法

  • 文艺、体育和特种工艺单位经未成年人的父母或者其他监护人同意，可以招用不满16周岁的专业文艺工作者、运动员

  • 用人单位自用工之日起即与劳动者建立劳动关系。试用期包含在劳动合同期限内，从用工之日起起算，无权由谁决定或约定

  • 试用期工资不得低于本单位相同岗位最低档工资或者劳动合同约定工资的百分之八十，并不得低于用人单位所在地的最低工资标准。

  • 试用期最长不超过六个月。劳动合同期限三个月以上不满一年的，试用期不得超过一个月；劳动合同期限一年以上不满三年的，试用期不得超过二个月；三年以上固定期限和无固定期限的劳动合同，试用期不得超过六个月。

  • 试用期只能约定一次，同一单位就算换岗位了也不能再有试用期。

民法典

  • 不满八周岁的未成年人为无民事行为能力人，由其法定代理人代理实施民事法律行为。无民事行为能力人实施的民事法律行为无效。
    • 不到8岁不能赠送财产，赠送行为无效
    • 无效的或者被撤销的民事法律行为自始没有法律约束力。
  • 八周岁以上的未成年人为限制民事行为能力人，实施民事法律行为由其法定代理人代理或者经其法定代理人同意、追认；但是，可以独立实施纯获利益的民事法律行为或者与其年龄、智力相适应的民事法律行为。
    • 到了8岁可以赠送财产，但赠送行为有没有效由爹妈说了算。取决于其法定代理人是否追认。

社会保险

  • 非本人意愿辞职不能领失业保险金（禁止白嫖）

  • 工伤保险判定：

    1. 工作时间
    2. 工作地点
    3. 因工作原因

公文

  • 决议↓
    • 适用于会议讨论通过的重大决策事项
    • 特点
      1. 权威性，一经公布，全党、全国上下必须执行
      2. 程序性，必须按法定会议法定程序讨论
      3. 指导性
    • 成文时间：决议正式通过的日期。标题下小括号注明会议名称及通过时间，也可以只写年月日
  • 决定↓
    • 适用于对重要事项做出决策和部署、奖惩相关、变更或撤销不适当的决定事项
    • 特点
      1. 规定性
      2. 指令性
      3. 稳定性
    • 根据内容和作用进行分类：
      1. 指挥性决定。对重大事项和行动做出的安排
      2. 知照性决定。用于机构设置、人事安排、撤销下级不适当决定等重要事项的宣告
      3. 奖惩性决定。表彰或处分
  • 命令↓
    • 适用于公布法规规章、实行重大强制性措施、批准授予和晋升衔级、嘉奖有关单位和人员
      • 需要注意区分的是：命令只奖不惩；而决定和通报可奖可惩。
      • 奖励对象关键词：“急难险重”的工作任务
    • 特点
      1. 权威：只有国家主席、全国人大常委会及委员长；国务院、总理、各部委、各部长、各委员会主任；地方人大代表大会委员会、县级以上地方各级人民政府，才可以在法定权限内发布命令。其他任何单位和个人无权发布。
        强制力，无条件执行
      2. 强制：无条件执行
      3. 严肃：不得随意更改或处理
      4. 限定：只限于重大事件，一般公务工作不得使用
    • 主要分类
      1. 任免令。仅限中央层面，其他级别不得使用
      2. 发布令。主席发布法律
      3. 行政令。公布重大强制性措施，例如《事业单位会计准则》
      4. 嘉奖令。是最高级别的奖励类型，表彰超人。
  • 公告↓
    • 适用于向国内外宣布重要事项或法定事项
    • 特点
      1. 限定：高层行政机关才能发布
      2. 广泛：范围面向国内外
      3. 内容和传播方式具有新闻性
    • 按内容分类
      1. 宣布重要事项的公告
      2. 宣布法定事项的公告
      3. 专业性公告
  • 公报↓
    • 适用于公布重要决定、重大事项
    • 特点
      1. 庄严：内容多是党内外、国内外普遍关心的重要事项
      2. 形式多样：可以通过新闻媒体刊发
      3. 公开
    • 按内容分类
      1. 新闻公报。以新闻形式在新闻媒体发布，阅知无限制，要求及时、真实
      2. 联合公报。政党、国家、政府等同层面之间，就某重大事项会议协商一致后，双方联合签署发布的文件
        • 联合公报有一些双方认可、联合签署的条文，比一般新闻公报具有更多具体内容
      3. 会议公报。党和国家重要会议情况发布的公报。例如《中国共产党xx届xx委员会全体会议公报》
      4. 统计公报。国家和政府统计机关发布的国民经济、社会发展方面情况的综合公报
  • 通告↓
    • 适用于在一定范围内公布应该遵守或周知的事项
    • 特点
      1. 广泛：使用频率和使用单位广泛
      2. 法规性
      3. 专业性：公安、交通、税务等，带有专门性质
    • 分为法规性通告和知照性通告
      1. 法规性通告强制性措施较多，违反将处罚
      2. 知照性通告目的是通知事项，没有强制性措施，受众只要接受所通知的事实
  • 通报↓
    • 适用于表彰、批评、传达精神、告知情况
      • 例如《xxx特大交通事故的通报》
    • 特点
      1. 教育
      2. 说理
      3. 时效
    • 分类
      1. 表彰性通报
      2. 批评性通报
      3. 情况通报
  • 报告↑
    • 向上级汇报、反馈、回复上级询问
    • 特点
      1. 陈述
      2. 沟通
      3. 单向：上级无需批复报告，避免在报告中向上级请求批示
      4. 可以一文多事
  • 请示↑
    • 适用于向上级请求指示、批准
    • 特点
      1. 单一：一事一文，一文一事
      2. 期复：必须回复
      3. 时效：搞快点给我批复
  • 批复↓
    • 适用于答复下级请示
    • 特点
      1. 公文中唯一被动的文种。下级有请示，上级才有批复。
      2. 针对：请示什么就批复什么
      3. 指导性
  • 议案↔
    • 适用于向同级提出审议事项
    • 特点
      1. 发文机关：各级人民政府（向同级人大或人常）
      2. 行文定向流动
      3. 时限性极强
  • 函↔
    • 适用于不相隶属机关之间商洽工作、询问、答复、请求批准、审批事项
    • 特点
      1. 行文方向多样性
      2. 使用的广泛性
      3. 写作的灵活性
    • 按性质分类（西装和便装）
      1. 公函。用于机关单位正式公务活动来往
      2. 便函。日常事务工作处理
    • 按内容和用途分类
      1. 商洽函。不相隶属机关之间商谈事宜
      2. 询问答复函。对公务中不确定的问题提出咨询、给予答复。没有固定方向，有无隶属关系皆可使用
      3. 求批审批函。向别家的老大提出请求批复或反过来答复审批
    • 按行文方向
      1. 发函。发起方，结束语一般为：此函询、特此函达、如蒙惠允、不胜感荷。
      2. 复函。答复方，结束语一般为：复函口、特此函复、特此复函。
  • 纪要↑↓↔
    • 适用于记载会议主要情况和议定事项
    • 特点
      1. 纪实：如实反应会议基本内容事项，未讨论议题、未达成共识的事项不能写
      2. 纪要：对会议内容进行归纳总结，记录主要事项，突出会议重点和主题，而不是有闻必录
      3. 约束：与其他行政公文一样，具有一定约束力
公告 公报 通告 通报
适用范围 向国内外宣布重要事项或法定事项 公布重要决定、重大事项 在一定范围内公布应该遵守或周知的事项 适用于表彰、批评、传达精神、告知情况
发布主体 高层行政机关才能发布 党政机关都可以使用 不限 不限
发布方式 报纸、广播、网络等媒体。但是不张贴 纸质发行、网上发布 各种媒体，张贴、悬挂均可 形式多样，可以是口头通报、电话通报、书面通报等
内容侧重 旨在让受众群体知晓遵守执行。公告篇幅短，简练准确、开门见山、直书其事。 旨在使大众了解。公报结构严谨、层次清晰，语言庄重热烈 公安、交通、税务等，带有专门性质 表彰、批评、传达精神、告知情况 - 例如《xxx特大交通事故的通报》
  • 公告与公报的区别：
    1. 发文机关：
      • 公告仅限制高层机关才能发布
      • 公报是政党机关都可以使用
    2. 受文范围：
      • 公告多为某领域政策、知识。可以是国内外机构组织个人。侧重于国家机关向国内外宣布重要事项或法定事项。
        • 例如，中国政府网公告主要用于宣布重要法定事项、规章制度、指示规定等，旨在让受众群体知晓遵守执行。
      • 公报更广泛，没有特定性和固定性，面向社会大众广而告之。公报侧重内容都是重大决策决定、过程经过、成效成果，向社会公开报道，旨在使大众了解。
        • 例如，会议公报主要用于介绍重要会议情况和公布会议结果；各省区市的经济运行公报主要用于公布有关经济发展数据；科研机构的科学公报主要用于公布重大科学研究成果。
    3. 表达形式：
      • 公告篇幅短，简练准确、开门见山、直书其事。
        • 例如，《新疆维吾尔自治区统计局第七次全国人口普查公告》，开篇明义，即对全区14个地州市及所辖县市区人口数据进行公布。相较于公报，公告就事论事，简短精练，不作阐述评论。
      • 公报结构严谨、层次清晰，语言庄重热烈
        • 例如，《中国共产党新疆维吾尔自治区第十届委员会第一次全体会议公报》，标题含义完整，主体内容详细，包含时间、地点、参加人员、会议内容、发布机构、公布日期等，依次介绍会议宗旨、主要议题、重要决策、取得成果、决定事项等。从语言风格来看，用词既庄重又通俗，语气既严肃又热烈，对会议的重大意义和产生的影响等进行阐述或评论。

未分类

  • 中华人民共和国主席根据全国人民代表大会的决定和全国人民代表大会常务委员会的决定，公布法律，任免国务院总理、副总理、国务委员、各部部长、各委员会主任、审计长、秘书长，授予国家的勋章和荣誉称号，发布特赦令，宣布进入紧急状态，宣布战争状态，发布动员令。

  • 行政许可，是指依法准予，case语句。具体是指行政机关根据公民、法人或者其他组织的申请，经依法审查，准予其从事特定活动的行为。”

    • 例如签发通行证、考取资格证书、审核确认资格

  • 行政确认，是指法律意义上的确定一件事情，global语句。具体是指行政主体依法对行政相对人的法律地位、法律关系或有关法律事实进行甄别，给予确定、认定、证明（或证伪）并予以宣告的具体行政行为。

    • 例如结婚
