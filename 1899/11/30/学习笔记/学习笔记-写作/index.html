学习笔记-写作

1899/11/30 学习笔记 学习笔记

菜就多练

题型概述

  1. 概括归纳类

    • 要素：表现（成绩、问题）、原因、影响（意义、危害）、对策
    • 答题方法：
      • 三删一改
        1. 删无关：与题干无关的删了不答。例如概括措施，就不能回答材料某某人说取得了什么什么成绩
        2. 删多余：修饰词、废话、重复，全部删掉做精简
        3. 删具体：具体的措施删改成笼统概括。
        4. 改不规范：具体的、口语化的要改。
      • 留意动词，动词一般会揭示措施。注意高频词，很可能是关键。
      • 过渡句、总领句、总结句、观点句
    • 作答要求：
      1. 直接加前置词（找主旨句）分条作答，同类合并，异类罗列
      2. 写短词短句

  2. 综合分析类

    • 要素：要素分析、词句理解、评价分析、对比变化
    • 答题方法：在概括的基础上进行合理推断。
      1. 概括：能抄先抄
      2. 推断：不能脱离材料，需要在材料中抓到可以推断的落脚点，不能凭空推断。措施、原因（问题）可以反说互推。
        • 例如措施：A村想要发展教育，当务之急就是拓宽经费来源。
          • 推断原因：经费来源狭隘
      3. 合理：不能天马行空。
    • 作答要求：
      1. 要素分析（例如分析原因）：跟概归纳一样，加前置词分条作答
      2. 词句理解
        1. 某概念的含义：概念表现 + 举例子
        2. 对“某词”做一全面解释：分段作答，看字数安排：表层概念 + 深层原因 + 解决方法
        3. 某句话的含义/理解：有抽象词的，分段解释抽象词的概念（看字数添加稍具体解释），最后用一句话总结整句话连在一起的含义。
      3. 评价分析：对资料中出现的观点、现象进行评价、谈看法。分三段作答
        1. 表达观点（表态）（简短表态）（是什么）
        2. 论证观点（论据）（分条作答）（为什么）
        3. 得出结论（结论）（押题论点）（怎么办，视材料和格子）
      4. 比较分析
        1. 按大结论分条回答：xx不同（大结论）：a如何（小结论）；b如何（小结论）

  3. 解决问题类

    • 单问：仅写对策
    • 多问：先写问题或原因，再写对策（简写问题，侧重对策，对策分配字数要更多）
    • 具体：做什么 + 怎么做
    • 针对性
    • 可行性

  4. 应用文写作

    • 常考应用文：公开信、倡议书、宣传稿、发言稿、讲解稿、新闻稿、新闻评论、汇报材料、情况反映、调查问卷、编者按、序言等

    • 主要格式包括：标题、称谓、正文、落款四大部分。

    • 格式要求

      1. 标题：一行居中，多行梯形或菱形，完整短词不拆分。除法规、规章名称加书名号外，一般不加标点符号。常见标题写法：
        1. 发文机关+关于+事由+的+文种：《国务院关于取消一批行政许可等事项的决定》
        2. 发文机关+文种：《中华人民共和国国家发展和改革委员会公告》
        3. 关于+事由+的+文种：《关于深化产教融合的若干意见》
          • 绝大部分应用文都可以使用
        4. 事由+文种：《共创全国文明城市倡议书》
        5. 仅文种：《倡议书》《通知》《公告》《发言稿》
          • 常见在非正式场合，例如小区公告栏、海报宣传栏等位置使用
        6. 仅事由（即文章主题）：《习近平会见巴拿马国民大会主席阿夫雷戈》《山东烟台一小区深夜犬吠 居民苦不堪言》
          • 一般用于新闻类、讲话类应用文。
        7. 题目给定标题
      2. 称谓：对同一主体不能并用各位和们
      3. 正文：结构层次序数依次可以用 一 （一） 1. （1） 标注。可跳用，不可反用。
      4. 落款：发文机关署名和成文日期。正文下一行右空二字书写

    • 一般分三段

      1. 第一段开门见山简明扼要，一般是背景、寒暄问候、解释说明。然后具体安排如下过渡到正事。
        • 如果是通知类的要注意绕个小弯，是为了提高xxx为了xx好才展开xxx培训，不然别人看了就不想参加。
      2. 第二段是正事，通知的具体详细安排，按结构层次分条
      3. 第三段结尾，希望式、请求式、模式化（以上模式请领导参阅）、希望各部门引起重视，强调主旨，也可以酌情省略如，“特此××”、“专此××”等
    • 文体类型 标题 称谓 落款
      短文、短评、报道、新闻稿、简报、导言、材料、方案等 √
      发言稿、讲话稿、讲解稿、演讲稿等 √ √
      宣传稿、倡议书、公开信、建议书、通知等 √ √ √

  5. 议论文写作

    1. 开篇立意
      1. 破题：梗概案例，或解释关键词句等，点名题意
        • 一般由题干破题，解释题干要求
      2. 承题：进一步分析影响、原因、启发
      3. 立意：提出自己的核心看法、观点
        • 立意要选大方向，分论点更容易扣题
    2. 论点论证
      1. 确立分论点
        1. 评论式：A是什么、B是什么、A与B的关系是什么，点明二者关系
        2. 矛盾统一：A是B的什么、B是A的什么、AB相结合的意义作用（AB对C的作用）
        3. 相互对立：A对B的影响、B对A的影响、打击A（解决问题），保护B（成绩）一肯一否
      2. 格式：分论点、（分析）、论据（实例）、分析论据（引析悟）、重申分论点

着力提升全民健身公共服务质量

（999字）

  1. 北京冬奥盛会成功举办，吸引更多人爱上冰雪运动；健身设施融入公园、广场等公共场地，居民身边的“15分钟健身圈”越来越多；赛事活动更加多样，攀岩、滑板等运动走进日常生活……随着我国深入实施全民健身国家战略，越来越多的人在运动中体验到乐趣。进一步提升全民健身公共服务质量，解决好广大群众就近健身等问题，成为新的社会需求。
    • 中心论点段
    • 全民健身国家战略
      • 就近健身
      • 提升全民健身公共服务质量
  2. 近年来，我国全民健身公共服务体系建设取得明显成效。截至2021年底，全国共有体育场地397.1万个，面积达34.1亿平方米，分别比2013年增长134.3%和71.2%。但是，与人民群众日益增长的健身需求相比，还存在着不足。比如，场地难求等问题依然存在，如何让健身设施更公平可及、更绿色便捷等课题，也有待破解。
    • 展开段
    • 全民健身公共服务体系建设
      • 公平可及
      • 绿色便捷
  3. 推动全民健身事业上新台阶，应注重解决公共服务资源供需错配的问题。必须坚持以人民为中心的发展思想，让全民健身公共服务资源跟着人转、跟着人走。让人口成为公共体育资源配置的“定位器”，是推动新时代全民健身工作的一个重要理念。一方面，把常住人口数量作为重要的资金分配考虑因素，优先在人口密度大、辐射范围广、体育设施供需矛盾突出的位置，建设公共体育设施。另一方面，引导优质资源向基层延伸推动全民健身公共服务城乡区域协调发展，让人们无论身处农村还是城市，都能享受大致均等的基本公共体育服务。
    • 分论点一：注重解决公共服务资源供需错配的问题
    • 以人民为中心
      • 常住人口数量
      • 优质资源向基层延伸
  4. 推动全民健身事业上新台阶，需精心打造绿色便捷的全民健身新载体。从导向上看，要推动健身场地设施更多向服务群众日常健身转变;从空间布局看，应拓展绿色生态的健身空间，让人们在亲近大自然的环境中开展体育锻炼。这些年，我国建了很多步道，群众使用热情很高。此外，滑雪、徒步、登山等户外运动的蓬勃发展，为群众参与健身活动创造了更好条件。
    • 分论点二：需精心打造绿色便捷的全民健身新载体
    • 新载体
      • 从导向上看
      • 从空间布局上看
  5. 推动全民健身事业上新台阶，还需加大公共体育设施的开放力度。2022年财政部将下达补助资金10.8亿元，用于公共体育场馆向社会免费或低收费开放。有的地方发布公共体育设施管理办法等规定，公共体育设施开放时间从每周不少于35小时延长到不少于70小时。全面推进公共体育场馆免费或低收费开放，推动学校体育场馆和社会体育场馆双向开放，将有效缓解健身需求与场馆资源供给不足的矛盾，让人民群众能就近享受更优质的健身场地供给。
    • 分论点三：还需加大公共体育设施的开放力度
    • 健身需求与场馆资源供给不足
      • “盘活存量”
      • 立法的角度
      • 具体管理的角度
  6. 全民健身事关全体人民的福祉。对个人而言，它是最实惠的健康投资，可以让我们精神与体魄都更加强健。对社会来讲，它是最普惠的公共服务，有助于全周期保障人民健康。顺应人民对高品质生活的期待，不断提供更好的全民健身公共服务，就一定能为推进健康中国建设注入澎湃动能。
    • 结尾点题
    • 健康中国建设
      • 普惠的公共服务
      • 全周期保障人民健康

打造年轻村干部队伍 筑牢基层党组织根基

（711字）

  1. 党的基层组织是党在社会基层组织中的战斗堡垒，是党的全部工作和战斗力的基础。当前，正值两个一百年奋斗目标的历史交汇期，脱贫攻坚取得全面胜利，乡村振兴推进如火如荼，新形势下建设更高水平的农村基层党组织和村干部队伍是关键之举。
  2. 突出头雁建设，优化基层党组织人员结构。“村民富不富，关键看支部；村子强不强，要看‘领头雁’。”村干部作为村级班子建设的主心骨，党员队伍的排头兵，队伍素质不仅直接影响到农村经济社会发展和社会和谐稳定，还影响到党在农村基层中的执政地位。基层党组织要想建设好、发挥好必须牢牢把握支部班子这个“关键少数”，各级党委政府要坚决贯彻党中央决策部署，鼓励和支持村党组织书记通过法定程序担任村委会主任和村级集体经济组织负责人，推动村“两委”班子成员交叉任职，杜绝党的领导与村民自治“两张皮”问题。要优选、高配、严管、厚爱“第一书记”队伍，求其实、尽其才，发挥好“第一书记”建强基层组织、推动精准扶贫、为民办事服务、提升治理水平等职责任务。
  3. 突出作用发挥，选优配强村干部队伍。选拔村干部队伍，注重在大中专院校毕业生和返乡创业青年中储备和培养村级后备力量，特别是注重培养村党组织书记后备人才。乡镇党委、村党组织应制定村级后备力量发展规划，切实解决村干部队伍存在的文化程度偏低、年龄老化和“后继乏人”等问题，引来“源头活水”。选优配强村干部队伍是发展壮大村集体经济，盘活资产资源，增加“造血”功能的必然要求，要建设一支懂经济、善经营、会管理、能力强的村干部队伍，发挥好基层党组织战斗堡垒作用，推动服务乡村振兴各项工作落地见效。
  4. 选好班子很重要，年轻干部不能少。面对年轻干部不是不愿意担任村干部，而是需要我们不断地给予扶持和帮助！
