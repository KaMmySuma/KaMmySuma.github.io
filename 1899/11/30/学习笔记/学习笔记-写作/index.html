学习笔记-写作

1899/11/30 学习笔记 学习笔记

菜就多练

题型概述

  1. 概括归纳类

    • 要素：表现（成绩、问题）、原因、影响（意义、危害）、对策
    • 答题方法：
      • 三删一改
        1. 删无关：与题干无关的删了不答。例如概括措施，就不能回答材料某某人说取得了什么什么成绩
        2. 删多余：修饰词、废话、重复，全部删掉做精简
        3. 删具体：具体的措施删改成笼统概括。
        4. 改不规范：具体的、口语化的要改。
      • 留意动词，动词一般会揭示措施。注意高频词，很可能是关键。
      • 过渡句、总领句、总结句、观点句
    • 作答要求：
      1. 直接加前置词（找主旨句）分条作答，同类合并，异类罗列
      2. 写短词短句

  2. 综合分析类

    • 要素：要素分析、词句理解、评价分析、对比变化
    • 答题方法：在概括的基础上进行合理推断。
      1. 概括：能抄先抄
      2. 推断：不能脱离材料，需要在材料中抓到可以推断的落脚点，不能凭空推断。措施、原因（问题）可以反说互推。
        • 例如措施：A村想要发展教育，当务之急就是拓宽经费来源。
          • 推断原因：经费来源狭隘
      3. 合理：不能天马行空。
    • 作答要求：
      1. 要素分析（例如分析原因）：跟概归纳一样，加前置词分条作答
      2. 词句理解
        1. 某概念的含义：概念表现 + 举例子
        2. 对“某词”做一全面解释：分段作答，看字数安排：表层概念 + 深层原因 + 解决方法
        3. 某句话的含义/理解：有抽象词的，分段解释抽象词的概念（看字数添加稍具体解释），最后用一句话总结整句话连在一起的含义。
      3. 评价分析：对资料中出现的观点、现象进行评价、谈看法。分三段作答
        1. 表达观点（表态）（简短表态）（是什么）
        2. 论证观点（论据）（分条作答）（为什么）
        3. 得出结论（结论）（押题论点）（怎么办，视材料和格子）
      4. 比较分析
        1. 按大结论分条回答：xx不同（大结论）：a如何（小结论）；b如何（小结论）

  3. 解决问题类

    • 单问：仅写对策
    • 多问：先写问题或原因，再写对策（简写问题，侧重对策，对策分配字数要更多）
    • 具体：做什么 + 怎么做
    • 针对性
    • 可行性

  4. 应用文写作

    • 常考应用文：公开信、倡议书、宣传稿、发言稿、讲解稿、新闻稿、新闻评论、汇报材料、情况反映、调查问卷、编者按、序言等

    • 主要格式包括：标题、称谓、正文、落款四大部分。

    • 格式要求

      1. 标题：一行居中，多行梯形或菱形，完整短词不拆分。除法规、规章名称加书名号外，一般不加标点符号。常见标题写法：
        1. 发文机关+关于+事由+的+文种：《国务院关于取消一批行政许可等事项的决定》
        2. 发文机关+文种：《中华人民共和国国家发展和改革委员会公告》
        3. 关于+事由+的+文种：《关于深化产教融合的若干意见》
          • 绝大部分应用文都可以使用
        4. 事由+文种：《共创全国文明城市倡议书》
        5. 仅文种：《倡议书》《通知》《公告》《发言稿》
          • 常见在非正式场合，例如小区公告栏、海报宣传栏等位置使用
        6. 仅事由（即文章主题）：《习近平会见巴拿马国民大会主席阿夫雷戈》《山东烟台一小区深夜犬吠 居民苦不堪言》
          • 一般用于新闻类、讲话类应用文。
        7. 题目给定标题
      2. 称谓：对同一主体不能并用各位和们
      3. 正文：结构层次序数依次可以用 一 （一） 1. （1） 标注。可跳用，不可反用。
      4. 落款：发文机关署名和成文日期。正文下一行右空二字书写

    • 一般分三段

      1. 第一段开门见山简明扼要，一般是背景、寒暄问候、解释说明。然后具体安排如下过渡到正事。
        • 如果是通知类的要注意绕个小弯，是为了提高xxx为了xx好才展开xxx培训，不然别人看了就不想参加。
      2. 第二段是正事，通知的具体详细安排，按结构层次分条
      3. 第三段结尾，希望式、请求式、模式化（以上模式请领导参阅）、希望各部门引起重视，强调主旨，也可以酌情省略如，“特此××”、“专此××”等
    • 文体类型 标题 称谓 落款
      短文、短评、报道、新闻稿、简报、导言、材料、方案等 √
      发言稿、讲话稿、讲解稿、演讲稿等 √ √
      宣传稿、倡议书、公开信、建议书、通知等 √ √ √

  5. 议论文写作

    1. 开篇立意
      1. 破题：梗概案例，或解释关键词句等，点名题意
        • 一般由题干破题，解释题干要求
      2. 承题：进一步分析影响、原因、启发
      3. 立意：提出自己的核心看法、观点
        • 立意要选大方向，分论点更容易扣题
    2. 论点论证
      1. 确立分论点
        1. 评论式：A是什么、B是什么、A与B的关系是什么，点明二者关系
        2. 矛盾统一：A是B的什么、B是A的什么、AB相结合的意义作用（AB对C的作用）
        3. 相互对立：A对B的影响、B对A的影响、打击A（解决问题），保护B（成绩）一肯一否
      2. 格式：分论点、（分析）、论据（实例）、分析论据（引析悟）、重申分论点

  • 逻辑论述

    • 三视青山【心态、多维视角】：
      1. 开头：辛弃疾有诗云:“我见青山多妩媚，料青山见我应如是”。我以仰视见青山，得清明高远；我以俯视见青山，得重晦深远；我以平视见青山，得明晦平远。见青山以得不同景，我心悠然自得。
      2. 分论点
        1. 仰视可观清明高远，不坠青云之志。
        2. 俯视可见重晦深远，以史为鉴明得失。
        3. 平视可睹明暗相映，得春山于平芜近处。
      3. 结尾：生逢盛世当欣逢盛世，作为新时代青年，既仰视立志，又俯视明史，同时平视而宁静致远，以此接过时代的接力棒，为人生而奋斗，三视青山，心远自得，以自得处人生，得平芜近处之青山。
    • 【青春、追梦】
      1. 开头：青裕之志，负履可行。理想，是熠熠生光的远山含着朝晖夕阳，是蓬勃而上的热烈带着豪情万丈。少年二字应与平庸相反，唯有理想，才能不顾一切。
      2. 结尾：”少年不知愁滋味，爱上层楼”，是青春少年大无畏攀登的勇敢；”仰天大笑出门去，我辈岂是蓬蒿人”，是青春少年初生牛犊傲视群雄的张狂。奋进新时代，逐梦青年”心之所向，素履以往”,”清澈的爱，只为中国”成为新时代青年的铮铮誓言。广大青年生逢其时、重任在肩，必须发愤图强，以只争朝夕的奋斗精神绘就青春亮丽的底色，把光和热洒满祖国大地，为第二个百年奋斗目标奉献自己的青春，创造出新的历史伟业！
    • 【坚持、毅力、挫折】
      1. 开头：放弃有十五笔，坚持有十六笔，失之一笔，差之千里，一切取决于你自己。山前有路，山后也未必没有曙光，所以请坚持下去，心中的火不能灭，哪怕别人只看得见烟。
      2. 结尾：梵高曾言：”首先是疼痛然后才是欢乐，唯其如此，则真理可喻功业可就。”贝多芬双耳失聪仍坚持狂热的音乐梦终扼住命运的咽喉；史铁生挣脱残疾的苦痛枷锁终成地坛之子；薇薇安摒弃他人嘲讽坚持自己终名声显赫。由此观之，若你能在一息尚存时坚毅摆渡，终会冲破风浪，仰望星空。纵有疾风起，人生不言弃！

      • 我们的直言义行中，是孔孟礼仪的绵延；我们的举手投足里，是魏晋风骨的接续；我们的衣食住行中，是华夏智慧的结晶。以深厚的文化为底色，才有今日的异彩纷呈，当我们骄傲地向世人展现中国风采，文化正是后盾。文化自信源于对本民族文化的深深依恋和认同，也源自以文化为支撑的家国发展。
    • 中国精神、爱国情怀
      • 精神所在，就是力量所在；血脉所系，就是动力所系。一百年前，中国共产党的先驱们创建了中国共产党，形成了坚持真理、坚守理想，践行初心、担当使命，不怕牺牲、英勇斗争，对党忠诚、不负人民的伟大建党精神。
    • 时代、担当、奋斗、使命、梦想
      • “一代人有一代人的长征，一代人有一代人的担当。”过往几代青年，皆以梦为马，以铁肩担道义，以不断探索中华之未来的脚步，铸就了如今这伟大的时代。青年的发展应该与时代环境同向而行，同频共振，时代昂扬向上的曲线，即是中国青年的生命轨迹。
      • 破茧成蝶是虫的梦想，日行千里是马的梦想，扶摇直上是鹏的梦想。因为有梦，我们攻坚克难，无畏挑战；因为有梦，我们努力追求、奋力拼搏。梦想是人生意义的源泉，是抵御虚无的归宿。梦想是云，让你不断向远方，不停追逐；梦想是心，让你不断看远方，持续守望。并不是所有的人都能达到梦想的彼岸，只因梦想是对勇敢者的奖赏。
    • 青春、砥砺奋斗
      • “世之奇伟、瑰怪，非常之观，常在于险远”，唯有以青春作注，不断探索开辟新发现，勇于走在时代前列，才能发现常人所目不及的新事物，才不会因虚度年华而悔恨，因碌碌无为而羞耻。岁月因青春慨然以赴而更加美好，世间因少年挺身而出而更加瑰丽。
      • 青春是我们共同的符号，奋斗是我们共同的心声，“中国青年”是我们共同的名字。想当年，面对国外技术封锁，一大批年轻科学家扎根荒漠，以“惊天一爆”撑起中国脊梁；面对赛场上的劲敌，中国女排以不服输的拼劲拼下“五连冠”，鼓舞亿万人民投身现代化建设；面对突如其来的河南汛情，无数“90后”“00后”勇赴防汛一线.一代代青年，把小我融入大我，以青春之光照亮大地色彩，皆因他们都是中国青年。

      • 不忘本来，才能开创未来。岁月峥嵘，山河为证；文脉悠远，与古为新。
    • 创新
