学习笔记-写作

1899/11/30 学习笔记 学习笔记

菜就多练

常考题型

  • 概括题

  • 公文写作

  • 大作文

  • 简答题（政治题）

  • 案例分析（概括材料、对策）

阅读方法

  • 关联词
    1. 因果关系：因为……所以，由于，因此
    2. 递进关系：还，不但…而且，进一步而言
    3. 并列关系：另外、首先、其次、最后；一方面……另一方面，既……又……
    4. 转折关系：虽然……但是，然而，其实，实际上，事实上
      • 转折之后是重点
      • 你人很好，但是我不喜欢
  • 首尾句
    • 通过阅读材料的首句、尾句来确定文章的主要内容、核心思想
    • 文章段落通常是：“总——分；分——总；总——分——总” 的形式，找总过分
  • 观点句
    • 材料中的观点性语句寻找作答要点
    • 材料某专家说的措施观点，可能就是答案
  • 高频词
    • 可以是近义词，不要局限完全一致的词语，例如延迟、迟延、低实时性

概括类写作

  • 题型特征
    1. 题干：概括、谈谈、总结、简述
      • 分析…的原因
      • 概括…的定义
      • 对…进行概述
      • 归纳…举例说明
      • 结合材料…为什么…？（概括原因）
    2. 要求：全面、准确、简洁、有条理
  • 作答
    1. 全面：多写不扣分，宁多不少、宁滥勿缺
      • 不要纠结，遇到像答案又不像答案的，无脑写上去
    2. 准确：没有歧义、优先使用原材料关键词
      • 优先用原材料的词，不要用自己的语言去臆测
      • 实在不行才自己提炼总结，注意表意不要违背材料
    3. 简洁：不写废话、规范表述
      • 材料中可能会出现冗杂的人物表述，例如，小张讲自己村里的变化，哗啦啦口语化的修饰词一大堆，这就要精简提炼。
        1. 例如，“大家腰包里有钱了，心气足了”、“隔壁大哥主动要求退出了低保”————总结提炼无非就是几个字“收入水平提高”
        2. 例如，大伙对生活讲究了起来，大妈有空就跳跳舞，大爷下下棋，哗啦哗啦一堆业余生活————总结提炼无非就是“精神文化生活需求增加”
        3. 例如，村里修了柏油路、有了路灯、哗啦啦一堆变化，生活那个方便啊————无非就是“基础建设更完善”
        4. 例如，隔壁村搞旅游的，修了排水沟、摆了垃圾箱、房子整一整，那个美啊————“产业多元化”、“村容村貌提高”
          • 旅游对应产业多元化
          • 环境对应村容村貌
      • 什么是废话？
        1. 文件、手册、政策等名称
          • 例如：为xxx发展，相关部门印发《阿姆斯特朗回旋倒吊大湾沟及拉呱里指导意见》，明确提出了几拉呱啦。
            • 概括措施就是——相关部出台文件。不要写那段超长的文件名，是废话
        2. “了、呢、吗”等结巴词
          • 材料中会出现这类词语，总结时应精简掉
            • 实现了xxx，总结为实现xxx；建立起了xxx，总结为建立xxx
        3. “等”之前的部分
          • 例如：村里公园安装了一批体育器材，如xx、xx、xx、xx、xx等
            • 总结：安装多种体育器材
            • 一批、xx、xx、xx都是废话
          • 例如：老王带干部考察十余次，选定了适合本村发展给的西瓜、二荆条辣椒和脆红李等三个主导项目
            • 总结：1、带干部实地考察 2、选定适合本村发展的项目
            • 十余次和那些个项目名字都是废话
        4. 修饰词
          • 例如：饥肠辘辘的我酣畅淋漓的吃了一堆大餐
            • 总结：我吃大餐
          • 政府 十分有规划地 大力 紧锣密鼓的 统统去掉
    4. 有条理：分条分点、分层分类
      • 按点答、分类答
  • 作答步骤
    1. 审题
      • 先读题，再根据提问找材料（阅读范围、核心事件、作答要素、字数）
    2. 勾画材料（问卷）
      • 用铅笔画？（待测试）
    3. 归类整理（问卷）
      • 意思一样的答案合并
      • 意思一样的答案分层
        • 总——分结构
        1. 按不同维度
          • 思想观念、政策制度、基础设施、监督管理、技术保障、收入水平
        2. 按不同主体
          • 个人、家庭、学校、企业、政府、社会组织、群众
        3. 按不同目的
          • 适用于找对策类的题目
          • 例如，村民对开发辣椒酱项目感到忧虑，于是干部小赵做了哪些哪些事，最终统一了大家的思想
            • 总括词：统一思想，打消顾虑：做法123。总分结构
        4. 按照材料逻辑☆
          • 材料可能是一段描写一个方法/要点，从段落中提取要点、从段落间分层
    4. 规范作答（答题卡）
      • 卷面整洁、字体干净。没啥好说的，字帖练字

对策题

  • 题型特征
    1. 题干：提出建议、措施
    2. 要求：定位准确、有针对性、分条作答
  • 作答
    1. 分层分点，总+分
    2. 简洁
  • 作答步骤
    1. 审题：根据哪个材料？站在谁的角度？提建议还是提措施？字数要求？
    2. 勾画：标注现成问题、对策；反推对策
    • 类似概括题

着力提升全民健身公共服务质量

（999字）

  1. 北京冬奥盛会成功举办，吸引更多人爱上冰雪运动；健身设施融入公园、广场等公共场地，居民身边的“15分钟健身圈”越来越多；赛事活动更加多样，攀岩、滑板等运动走进日常生活……随着我国深入实施全民健身国家战略，越来越多的人在运动中体验到乐趣。进一步提升全民健身公共服务质量，解决好广大群众就近健身等问题，成为新的社会需求。
    • 中心论点段
    • 全民健身国家战略
      • 就近健身
      • 提升全民健身公共服务质量
  2. 近年来，我国全民健身公共服务体系建设取得明显成效。截至2021年底，全国共有体育场地397.1万个，面积达34.1亿平方米，分别比2013年增长134.3%和71.2%。但是，与人民群众日益增长的健身需求相比，还存在着不足。比如，场地难求等问题依然存在，如何让健身设施更公平可及、更绿色便捷等课题，也有待破解。
    • 展开段
    • 全民健身公共服务体系建设
      • 公平可及
      • 绿色便捷
  3. 推动全民健身事业上新台阶，应注重解决公共服务资源供需错配的问题。必须坚持以人民为中心的发展思想，让全民健身公共服务资源跟着人转、跟着人走。让人口成为公共体育资源配置的“定位器”，是推动新时代全民健身工作的一个重要理念。一方面，把常住人口数量作为重要的资金分配考虑因素，优先在人口密度大、辐射范围广、体育设施供需矛盾突出的位置，建设公共体育设施。另一方面，引导优质资源向基层延伸推动全民健身公共服务城乡区域协调发展，让人们无论身处农村还是城市，都能享受大致均等的基本公共体育服务。
    • 分论点一：注重解决公共服务资源供需错配的问题
    • 以人民为中心
      • 常住人口数量
      • 优质资源向基层延伸
  4. 推动全民健身事业上新台阶，需精心打造绿色便捷的全民健身新载体。从导向上看，要推动健身场地设施更多向服务群众日常健身转变;从空间布局看，应拓展绿色生态的健身空间，让人们在亲近大自然的环境中开展体育锻炼。这些年，我国建了很多步道，群众使用热情很高。此外，滑雪、徒步、登山等户外运动的蓬勃发展，为群众参与健身活动创造了更好条件。
    • 分论点二：需精心打造绿色便捷的全民健身新载体
    • 新载体
      • 从导向上看
      • 从空间布局上看
  5. 推动全民健身事业上新台阶，还需加大公共体育设施的开放力度。2022年财政部将下达补助资金10.8亿元，用于公共体育场馆向社会免费或低收费开放。有的地方发布公共体育设施管理办法等规定，公共体育设施开放时间从每周不少于35小时延长到不少于70小时。全面推进公共体育场馆免费或低收费开放，推动学校体育场馆和社会体育场馆双向开放，将有效缓解健身需求与场馆资源供给不足的矛盾，让人民群众能就近享受更优质的健身场地供给。
    • 分论点三：还需加大公共体育设施的开放力度
    • 健身需求与场馆资源供给不足
      • “盘活存量”
      • 立法的角度
      • 具体管理的角度
  6. 全民健身事关全体人民的福祉。对个人而言，它是最实惠的健康投资，可以让我们精神与体魄都更加强健。对社会来讲，它是最普惠的公共服务，有助于全周期保障人民健康。顺应人民对高品质生活的期待，不断提供更好的全民健身公共服务，就一定能为推进健康中国建设注入澎湃动能。
    • 结尾点题
    • 健康中国建设
      • 普惠的公共服务
      • 全周期保障人民健康

打造年轻村干部队伍 筑牢基层党组织根基

（711字）

  1. 党的基层组织是党在社会基层组织中的战斗堡垒，是党的全部工作和战斗力的基础。当前，正值两个一百年奋斗目标的历史交汇期，脱贫攻坚取得全面胜利，乡村振兴推进如火如荼，新形势下建设更高水平的农村基层党组织和村干部队伍是关键之举。
  2. 突出头雁建设，优化基层党组织人员结构。“村民富不富，关键看支部；村子强不强，要看‘领头雁’。”村干部作为村级班子建设的主心骨，党员队伍的排头兵，队伍素质不仅直接影响到农村经济社会发展和社会和谐稳定，还影响到党在农村基层中的执政地位。基层党组织要想建设好、发挥好必须牢牢把握支部班子这个“关键少数”，各级党委政府要坚决贯彻党中央决策部署，鼓励和支持村党组织书记通过法定程序担任村委会主任和村级集体经济组织负责人，推动村“两委”班子成员交叉任职，杜绝党的领导与村民自治“两张皮”问题。要优选、高配、严管、厚爱“第一书记”队伍，求其实、尽其才，发挥好“第一书记”建强基层组织、推动精准扶贫、为民办事服务、提升治理水平等职责任务。
  3. 突出作用发挥，选优配强村干部队伍。选拔村干部队伍，注重在大中专院校毕业生和返乡创业青年中储备和培养村级后备力量，特别是注重培养村党组织书记后备人才。乡镇党委、村党组织应制定村级后备力量发展规划，切实解决村干部队伍存在的文化程度偏低、年龄老化和“后继乏人”等问题，引来“源头活水”。选优配强村干部队伍是发展壮大村集体经济，盘活资产资源，增加“造血”功能的必然要求，要建设一支懂经济、善经营、会管理、能力强的村干部队伍，发挥好基层党组织战斗堡垒作用，推动服务乡村振兴各项工作落地见效。
  4. 选好班子很重要，年轻干部不能少。面对年轻干部不是不愿意担任村干部，而是需要我们不断地给予扶持和帮助！
